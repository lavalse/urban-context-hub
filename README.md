# urban-context-hub

A prototype platform for managing urban operational events using FIWARE NGSI-LD.

## Modules

- `domains/*` - domain models and JSON-LD @context definitions
- `packages/*` - shared libraries (e.g., NGSI-LD client)
- `apps/api` - application backend (REST API for UI and integrations)
- `apps/web` - web UI
- `infra` - local infrastructure (Orion-LD + MongoDB)
- `docs` - architecture and lifecycle docs
